Predefined Classes

Because some patterns are used over and over again, a set of predefined characcters classes is used to make it easy for you to specify some complex classes. The following table lists all the predefined classes.

Code Equal To Matches
. [^\n\r] Any Character except new line and carriage return.
\d [0-9] A digit
\D [^0-9] A non digit
\s [ \t\n\x0B\f\r] A white space character
\S [^\t\n\x0B\f\r] A non white space character
\w [a-zA-z_0-9] A word character (all letters, all numbers and an underscore)
\W [^a-zA-Z_0-9] A non word character
