Range Classes

Suppose that you want to match any alphabet character, but you really dont want to type every letter in the alphabet Instead, you can use a range class to specify a range between a-z. The key here is a dash(-), which should be reading as through instead of minus(-).

Note that [a-z] matches only the lower case letters unless the regular expression is set to case sensitive by using the i-option. To match only upper case letters you must use [A-Z].
