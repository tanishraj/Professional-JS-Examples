Getting Form References

Before scripting a form, you first must get a reference to the form element. This can be done in a number of different ways.

Different ways of getting form references:

  • var oForm = document.getElementById("form1");
  • var oForm = document.forms[0]; //get the first form
  • var oOtherForm = document.forms["formZ"]; //get the form whose name is “formZ”
