Declaration And Instantiation

Objects are created by using 'new' keyword followed by the name of class you wish to instantiate.
Example:
var oObject = new Object();
var oStringObject = newString();

The first line creates the instance of object and stores it in the variable oObject. The second line creates a new instance of String and stores it in the variable oStringObject. The parentheses are not required when the constructor does not required arguments, so the above two lines can be rewritten as follows...
Example:
var oObject = new Object;
var oStringObject = newString;
