Keyboard Events

Keyboard events are caused by user action on the keyboard. The keyboard events are the following:

  • keydown — Occurs when the user presses a key on the keyboard. It also occurs repeatedly as the key is being held down.
  • keypress — Occurs when the user presses a key on the keyboard that results in a character (disregards keys like Shift and Alt). It also occurs repeatedly as the key is being held down.
  • keyup — Occurs when the user releases a key that was down.
