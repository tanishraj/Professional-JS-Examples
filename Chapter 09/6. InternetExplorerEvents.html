DOM Event Handler

In IE, every element and window object has two methods: attachEvent() and detachEvent(). As the names indicates, attachEvent() is used to attach an event handler to an event and detachEvent() is used to detach an event handler. Each method takes two arguments: the name of the event handler to assign to (for example: onclick) and a function.

Syntax:
[Object].attachEvent(“name_of_event_handler”, fnHandler);
[Object].detachEvent(“name_of_event_handler”, fnHandler);

In the case of attachEvent(), the function is added as an event handler; for detachEvent(), it looks for the given function in the event handler list and removes it.
