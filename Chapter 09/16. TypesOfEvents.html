Types of Events

The events that occur in a browser can be grouped into several specific types, depending on the object the event is fired from and what triggered the event to fire. The DOM specification defines the following event groups:

  • Mouse Events are fired when the user uses the mouse to perform certain actions.
  • Keyboard Events are fired when the user types on the keyboard.
  • HTML Events are fired when certain changes occur to the browser window or specific client server interaction occurs.
  • Mutation Events are fired when a change occurs to the underlying DOM structure.
