Scripting Styles

When Cascading Style Sheets (CSS) were introduced 1996, they completely changed the way developers formatted their HTML pages. Instead of using HTML tags such as and , pages began using CSS to define the appearance of fonts and other items. The natural next step for CSS support was to make styles accessible from JavaScript.

Internet Explorer 4.0 introduced a style object for each element on a page to manage that element’s CSS-defined styles. The DOM eventually adopted this approach as a standard way to access an element’s style information.
