System Dialogs

Aside from popping up new browser window, several other pop up information to the user utilizing method of the window object. alert(), confirm(), prompt().

  • alert(): alert the message.
  • confirm(): It is also similar to alert by it has two button OK and CANCEL. Once you select the OK button operation will be performed and when you are choosing CANCEL button it terminate the process then and there.
  • prompt(): It is very useful if you want to take a input from user at runtime.
