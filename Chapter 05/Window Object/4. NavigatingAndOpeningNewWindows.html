Navigating And Opening New Windows

Using JS, it is possible to navigate to URLs and open new browser windows using the window.open() method. This method accepts four arguments: the URL of the page to load in the new window, the name of the new window, a string of features, and a boolean value indicating whether the loaded page should take the place of the currently loaded page.

Attributes we can apply

  • left: Indicate the left co-ordinate of the new window. This can not be negative number.
  • top: Indicate the top co-ordinate of the new window. This can not be a negative number.
  • height: set the height of the new window. This can not be a number less than 100.
  • width: set the width of the new window. This can not be less than 100.
  • resizable: determines if the window can be resized by dragging on its border. The default is no.
  • scrollable: determines if the window allows scrolling if the content can not fit in the viewport. The default is no.
  • toolbar: determines if the new windows has its own toolbar. The default is no.
  • status: Determines if the new window has its status bar showing. The default is no.
  • location: determines if the new window has its location showing. The default is no.
