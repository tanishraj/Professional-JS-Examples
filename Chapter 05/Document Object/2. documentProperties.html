document Properties

document object has got lots of properties which are given below...

Properties:

  • alinkColor: The color for active links as defined by the body.
  • bgColor: The color for the page background as defined by body.
  • fgColor: The color for text as defined by body.
  • lastModified: The date the page was last modified as a string.
  • linkColor: The color for links as defined by body.
  • referrer: The URL one position back in the browser history.
  • title: The text displayed in the title tag.
  • URL: The url of the currently loaded page.
  • vlinkColor: The color of the visited link as defined by the body.
