Creating And Manipulating Nodes

The DOM document has a number of methods designed to create various types of nodes, even though the browser document object does not necessarily support each of these methods in all browsers. Methods are listed below....

Methods:

  • createAttribute(text): create an attribute name with the given name
  • createCDATASection(text): creates a CDATA section with the text child node containing text
  • createDocumentFragment(): create a document fragment node
  • createElement(tag name): creates an element with a tag name of tag name
  • createEntityReference(name): creates an entity reference node with the given name
  • createProcessingInstruction(target, data): creates a PI node with the given target and data
  • createTextNode(text): creates a text node containing text
